## Install

```
yarn install
npm run start
```
Run: ===> 
docker build -t familyshare-fe . && docker run -t -i -p 5002:80 --rm familyshare-fe. 
  
Front End features:
   - mui react design
   - map view for the children moves: ===> we use Pigeon Maps
   - map view for children in each activity
   - ring countdown timer for each activity
